About Cirque D'Hiver

Cirque d'Hiver is a website created to fulfill the learning goals of MSC2006 (and coincidentally the goals of its creator, Winter Kraemer). This site is an experiment in HTML, CSS, and maybe even JavaScript (but let's not get too crazy).


My main goals for this site:

  • Learn the basics of CSS, HTML, and JS
  • Create a site with a consistent clown theme (so far so good)
  • Understand linking and site structure to support my own MRP work

Uncertainties
